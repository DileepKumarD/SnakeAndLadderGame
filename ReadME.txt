**This is a visual studio Project**

To open this project in Visual Studio:
-Click on SnakeAndLadder\SnakeAndLadder.sln

To run the output directly run SnakeAndLadder\Debug\SnakeAndLadder.exe

To view the code files:
Go to SnakeAndLadder\SnakeAndLadder

      Game.cpp - This file contains the main function which starts the execution
      SnakeAndLadder.h
      SnakeAndLadder.cpp
      GameHandler.h
      GameHandler.cpp
      Player.h
      Player.cpp
